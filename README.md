# AI Development Workflow Assignment

This repository contains code and documentation for the AI Development Workflow assignment for the course "AI for Software Engineering".

## Structure
- `data/` - Raw and processed datasets
- `notebooks/` - Jupyter notebooks for analysis and modeling
- `src/` - Python scripts for preprocessing, modeling, and evaluation
- `figures/` - Workflow diagrams
- `report/` - PDF report for submission

## Instructions
1. Install dependencies: `pip install -r requirements.txt`
2. Run notebooks in order:
   - Part1_Short_Answers.ipynb
   - Part2_Case_Study.ipynb
3. Preprocessed data and trained models are saved in `data/processed/` and `models/` (if any)
